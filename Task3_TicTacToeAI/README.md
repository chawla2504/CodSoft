# Tic-Tac-Toe AI

## Overview

This project implements an AI-powered Tic-Tac-Toe game using the Minimax Algorithm. The AI evaluates all possible game states and selects the optimal move, making it difficult to defeat.

## Technologies Used

- Python

## Features

- Human vs AI gameplay
- Minimax Algorithm
- Optimal decision making
- Terminal-based interface

## Run

```bash
python tic_tac_toe.py
```

## AI Concept

The Minimax Algorithm is used to simulate future game states and choose the move that maximizes the AI's chances of winning while minimizing the opponent's chances.
